# Raw Knowledge Dump Folder

Drop raw content here before ingesting.

## File Naming

```
YYYY-MM-DD-{short-desc}.md
```

Examples:
- `2026-04-16-wiki-graph-pattern.md`
- `2026-04-16-obsidian-second-brain.md`

## Template

```markdown
# {Title}

Source: {URL or description}
Date: YYYY-MM-DD
Tags: {tag1}, {tag2}

## Content

{paste raw content here}

## Key Concepts (optional — AI will extract if empty)

-
```

## After Dropping File

Tell AI: `"ingest raw/YYYY-MM-DD-{desc}.md"`
