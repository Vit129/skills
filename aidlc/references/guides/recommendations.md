# Recommendations & Decision Framework

## Decision Framework

For each decision point, provide:
1. Recommended Approach with specific reasoning
2. Alternative Options with pros/cons
3. Decision Factors (timeline, complexity, team skills)
4. Risk Assessment and mitigation strategies

## Principles

- Always provide specific recommendations — never ask "what should we do?"
- Include reasoning — explain why one option is preferred
- Present alternatives — give users informed choices
- Consider context — team skills, timeline, complexity
- Be decisive — have a clear preferred approach
- NEVER fill decision answers — always leave blank for user
