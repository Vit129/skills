# Tax Optimizer Skill

วิเคราะห์และแนะนำกลยุทธ์ลดภาษีให้เหลือน้อยที่สุดผ่าน RMF/SSF/ThaiESG

## ข้อมูลที่ต้องการ

ดึงจาก TaxInputPanel หรือถามผู้ใช้ถ้ายังไม่มีข้อมูล:
- รายได้รวมปีนี้ (THB)
- ค่าลดหย่อนที่ใช้ไปแล้ว (ประกันสังคม, ประกันชีวิต ฯลฯ)
- ยอด RMF/SSF/ThaiESG ที่ซื้อไปแล้ว

## ขั้นตอน

### 1. คำนวณ Tax ปัจจุบัน
ใช้ tax brackets จาก `tax-review` skill คำนวณภาษีก่อนซื้อกองทุนเพิ่ม

### 2. หา Optimal Purchase Amount
- คำนวณ marginal tax rate ปัจจุบัน
- หาว่าซื้อ RMF/SSF/ThaiESG เพิ่มเท่าไหร่จึงจะผ่าน tax bracket ลงไปอีกขั้น
- ตรวจสอบ cap: รวม RMF+SSF+ThaiESG+PVD ต้องไม่เกิน 500,000 THB และ ≤30% ของรายได้

### 3. คำนวณ ROI ของการซื้อกองทุน

```
ซื้อ RMF X บาท → ลดหย่อนได้ X บาท → ประหยัดภาษี X × marginal_rate
ROI = ประหยัดภาษี / เงินที่ลงทุน × 100%
```

   ROI: XX% (ได้เงินคืนจากภาษีทันที ยังไม่นับผลตอบแทนกองทุน)
   ภาษีที่ต้องจ่าย: ฿XX,XXX
