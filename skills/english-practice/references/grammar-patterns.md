# English Practice — Grammar Patterns Reference

## Past Tense
- **Regular:** add -ed (walk → walked)
- **Irregular:** change form (write → wrote, go → went)
- **When to use:** completed actions, past habits

## Present Perfect (have + past participle)
- **Form:** have/has + verb-ed/irregular
- **When:** recent action with current relevance ("I've fixed the bug" = it's fixed now)
- **vs Past:** "I fixed it yesterday" = completed past; "I've fixed it" = current result matters

## Articles (a/an/the)
- **a/an:** first mention of countable noun (a bug, an error)
- **the:** specific or previously mentioned (the bug I found yesterday)
- **No article:** uncountable nouns, plural generics (I like coffee; bugs are common)

## Gerunds (-ing) vs Infinitives (to + verb)
- **Gerund:** "I enjoy writing code" (the activity itself)
- **Infinitive:** "I want to write code" (the action/goal)
- **Remember:** "I like writing" but "I want to write"

## Cause-Effect Sentences
- **Wrong:** "I think because I translate from Thai..."
- **Correct:** "I think it's because I translate from Thai..."
- Rule: "it's because..." for cause-effect; "because" alone feels incomplete as a standalone clause

## Thai Speaker Challenges

| Challenge | Thai Habit | English Correct |
|-----------|-----------|-----------------|
| Articles | Thai has none — skip them | "an engineer", "the bug" — never skip |
| Tense | Thai marks time with words, not verb form | Change the verb: wrote, will write |
| Prepositions | Thai word order differs | Trust English patterns, not literal translation |
| Passive voice | Thai uses it differently | Practice active voice first |
| Word order | Don't translate word-by-word | Learn English phrase patterns separately |

**Strategy:** When translating word-by-word from Thai, STOP. Rewrite thinking in English sentence structure.
