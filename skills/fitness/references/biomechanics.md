# Reference: Biomechanics & Movement Modification (Unit-Aware)

## Unit System Configuration

**Load measurements adapt to user preference:**

### Imperial (USA/UK)
* Weight in lbs (pounds)
* Load increments: +5–10 lbs per session
* Example: Start 135 lbs → Add 5 lbs → Next session 140 lbs

### Metric (International)
* Weight in kg
* Load increments: +2.5–5 kg per session
* Example: Start 60 kg → Add 2.5 kg → Next session 62.5 kg

### Auto-Conversion Reference
* 1 kg = 2.2 lbs
* 1 lb = 0.453 kg
* 2.5 kg ≈ 5.5 lbs
* 5 kg ≈ 11 lbs

**System Behavior:**
* User selects: Imperial or Metric
* AI displays all loads in selected unit
* AI converts pain thresholds, ROM, angles (unit-independent)

## Core Principles
* Assess pain location, ROM limitations, and structural constraints BEFORE recommending modifications
* Safe angle ranges take precedence over exercise variety
* Use unilateral (single-limb) options to bypass bilateral limitations
* Progress load incrementally while monitoring pain feedback

## Structural Limitation Protocols

### Right Shoulder — Sub-Acromial Impingement Management
**Safe ROM:** 0–120° flexion, limit horizontal adduction
**Avoid:** Overhead press, behind-neck pull, heavy bench press with wide grip

**Modifications:**
* Incline press (30–45°) instead of flat bench
* Neutral-grip dumbbell press (palms facing each other)
* Landmine press (single-arm, pressing into ground-anchored bar)
* Seated cable row (emphasize scapular retraction)
* Machine-based shoulder work (fixed path = safer ROM)

**Progression:**
1. Pain-free ROM with light load (5–10 reps)
2. Add load per session (metric: 2.5–5 kg | imperial: 5–10 lbs)
3. When pain-free for 3 consecutive sessions → advance to next variation

**Example (Metric):** 20 kg → 22.5 kg → 25 kg → 27.5 kg
**Example (Imperial):** 45 lbs → 50 lbs → 55 lbs → 60 lbs

---

### Right Hip — External Rotation Restriction
**Safe Movement:** Hip flexion, abduction, internal rotation preferred
**Avoid:** Forced external rotation, deep squats with toes-out, pigeon pose

**Modifications:**
* **Lower Body:** Staggered stances (offset legs) for compound movements
* **Single-leg focus:** Bulgarian split squats (rear leg elevated)
* **Knee-dominant:** Leg press with neutral foot angle
* **Hip-hinge:** Deadlifts with feet hip-width, minimal external rotation

**Progression:**
1. Bodyweight variations (3 sets × 8–12 reps)
2. Add load gradually (2.5–5 kg increases)
3. Monitor for pain onset; reduce load if pain appears

---

### Right Leg Length Discrepancy (LLD) — Pelvic Stability Protocol
**Mechanism:** Shorter leg + bilateral loading = pelvic tilt, spinal stress
**Solution:** Asymmetrical setup or heel elevation

**Modifications:**
* **Bilateral squats:** Place 5–10 mm shim under shorter leg's heel
* **Deadlifts:** Offset stance (shorter leg slightly forward)
* **Single-leg work:** Emphasize shorter leg in unilateral exercises
* **Assessment:** Use mirror or video to confirm neutral pelvic alignment

**Progression:**
1. Master shim-assisted form first
2. Gradually reduce shim thickness over 4–6 weeks
3. Reassess pelvic alignment; return to shim if tilt returns

## Exercise Form Checklist

**Before teaching any exercise:**
1. ✅ Assess current ROM + pain points
2. ✅ Identify safe angle ranges for this client
3. ✅ Explain WHY modification is needed (not a limitation, a strategy)
4. ✅ Demonstrate correct form with modification
5. ✅ Watch 2–3 reps; provide real-time feedback
6. ✅ Progress load only if form stays consistent

---

## Pain vs. Discomfort
* **Discomfort:** Muscle fatigue, mild muscle soreness = OK to push
* **Pain:** Sharp, pinching, radiating = STOP immediately, regress exercise

**Report:** "Discomfort noted—monitor for increased pain" or "Pain onset detected—switch to modified variation"
