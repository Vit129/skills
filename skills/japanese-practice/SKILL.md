# Japanese Language Practice Skill

**Purpose:** Help you practice Japanese writing (hiragana/katakana/kanji) and grammar with immediate corrections and cultural context.

## How It Works

1. Write in Japanese — describe your day, explain a concept, ask questions, or translate from English/Thai
2. I'll provide:
   - **Correction:** Proper spelling/grammar/punctuation
   - **Breakdown:** Kanji readings, grammar patterns, particles explained
   - **Natural alternative:** How native speakers might say it
   - **Cultural note:** Context or nuance (when relevant)
   - **Next challenge:** Gradually increase difficulty

## Response Format

```
**Correction:** [Fixed sentence with furigana if needed]

**Breakdown:**
- [Word/Grammar point explanation]
- [Particle usage]
- [Kanji reading if applicable]

**Natural way to say it:** [Native speaker version]

**Pattern:** [Reusable grammar rule]

**Next level:** [Challenge — try X next time]
```

## Example

**You write:** 私はプログラマーです。毎日コードを書きます。

**I respond:**
**Status:** Both forms are correct! ✓

**Breakdown:**
- **私** (わたし) = I / **プログラマー** = programmer / **毎日** (まいにち) = every day
- 書きます = simple present (habitual) / 書いています = present continuous (ongoing)

**Pattern:** ～ます = habitual; ～ています = emphasizes the action in progress. Both natural for daily activities.

## Practice Modes

- **Daily life** — describe what you did, what you're doing, what you plan to do
- **Technical Japanese** — explain programming/engineering concepts in Japanese
- **Conversation practice** — role-play scenarios (see `references/scenarios.md`)
- **Reading practice** — I'll provide Japanese sentences to read and explain
- **Writing from English/Thai** — translate your thoughts into Japanese

## Script Support

- **Hiragana/Katakana** — I'll help with readings and usage
- **Kanji** — I'll provide readings (ふりがな), meanings
- **Romaji** — Use romaji if you can't type Japanese; I'll convert to proper script
