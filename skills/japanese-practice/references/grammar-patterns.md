# Japanese Practice — Grammar Patterns Reference

## Polite Request Forms

### Pattern 1: ～てくれませんか (Can you [do this] for me?)
```
読んで（よんで）くれませんか？  →  Can you read it for me?
教えてくれませんか？            →  Can you teach me?
手伝ってくれませんか？          →  Can you help me?
```
- Take verb in て-form: 読む → 読んで
- Add くれませんか (polite request to someone's action benefiting you)

### Pattern 2: ～くださいませんか (Could you please [do this]?)
```
読んでくださいませんか？         →  Could you please read it?
教えてくださいませんか？         →  Could you please teach me?
説明（せつめい）してくださいませんか？ →  Could you please explain?
```
- More formal than ～てくれませんか

### Pattern 3: ～てもらえますか (Can I get you to [do this]?)
```
読んでもらえますか？  →  Can I get you to read it?
直してもらえますか？  →  Can I get you to fix it?
```
- Slightly more casual than くださいませんか

**Comparison:**
| Pattern | Formality | Nuance |
|---------|-----------|--------|
| ～てくれませんか | Casual polite | "Can you do this for me?" |
| ～てもらえますか | Neutral | "Can I receive the favor?" |
| ～てくださいませんか | Formal | "Could you please?" |

## Kanji Reading Guide

**⚠️ This skill uses ふりがな (hiragana above kanji) ONLY — no romaji.**

```
漢字（かんじ）← kanji
      ↑ furigana (hiragana pronunciation)
```

How to read: ignore the kanji, read the hiragana: か(ka) ん(n) じ(ji) = "kanji"

## Common Error Examples (From Sessions)

### Mistake: Mixed request pattern
```
❌ 漢字の上のふりがなを読んで方ままえんか
✓  漢字（かんじ）の上（うえ）のふりがなを読んでくれませんか？
```
- 方ままえんか = not a real form
- Breakdown: 漢字の上の = above the kanji / ふりがなを = furigana (object) / 読んで = te-form of 読む / くれませんか = can you for me?

### Tense pattern: ～ます vs ～ています
```
書きます     = habitual or general present (I write code)
書いています = present continuous (I am writing code now)
```
Both are natural for daily activities — choose based on nuance.

Combined: 毎日コードを書きますが、今日は特に複雑なプロジェクトを書いています。
(I write code daily, but today I'm working on something particularly complex.)
