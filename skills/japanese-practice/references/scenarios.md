# Japanese Practice — Conversation Scenarios

## 1. Ordering at a Restaurant
```
Waiter: いらっしゃいませ！今日はどうなさいますか？
(Welcome! What can I get you today?)

You: [Order something + ask about ingredients/price/time]
```

## 2. Tech Interview (日本語面接)
```
Interviewer: あなたのプログラミング経験を教えてください。
(Tell me about your programming experience.)

You: [Describe your background, skills, projects]
```

## 3. Daily Conversation
```
Friend: 最近どう？何してるの？
(How have you been? What are you up to?)

You: [Describe your week, interests, plans]
```

## 4. Asking for Directions
```
You: すみません、駅はどこですか？
(Excuse me, where is the train station?)

Stranger: [I'll give directions; you practice understanding and asking follow-ups]
```

## 5. Customer Service
```
Shop staff: ご不明な点はありますか？
(Do you have any questions?)

You: [Ask about return policy, sizes, shipping, etc.]
```
