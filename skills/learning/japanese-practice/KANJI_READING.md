# How to Read Kanji Using Hiragana / Katakana (ふりがな)

## What is ふりがな (Furigana)?

**ふりがな** = small hiragana/katakana written *above* (in horizontal text) or *beside* (in vertical text) kanji to show how to pronounce it.

Example:
```
漢字（かんじ）
↑ kanji  ↑ furigana (hiragana above)

字（じ）
↑ kanji ↑ furigana showing pronunciation "じ"
```

---

## How to Read Kanji With Furigana

### Step 1: Identify the Kanji
Look for the Chinese character(s). In this skill, I'll provide furigana like this:

```
漢字（かんじ）
```

**What this means:**
- **漢字** = the kanji (Chinese characters)
- **（）** = parentheses with hiragana inside
- **かんじ** = the hiragana pronunciation (read as "ka-n-ji")

### Step 2: Read the Hiragana Instead
When you see:
```
漢字（かんじ）
```

You **say: かんじ** (ka-n-ji) — ignore the kanji, read the hiragana.

### Step 3: Understand the Structure

**Hiragana breakdown of かんじ:**
- か = "ka" sound
- ん = "n" sound (nasal)
- じ = "ji" sound

**Together:** かんじ = kan-ji = "kanji"

## Common Kanji Reading Patterns

### Pattern 1: Single Kanji + Single Hiragana Reading

```
字（じ）      → read as: じ
人（ひと）    → read as: ひと
本（ほん）    → read as: ほん
```

### Pattern 2: Multiple Kanji + One Hiragana Reading

```
漢字（かんじ）  → read as: かんじ (both kanji = one word)
日本（にほん）  → read as: にほん
木曜日（もくようび）  → read as: もくようび
```

### Pattern 3: Compound Words (カタカナ for foreign words)

```
コンピュータ（コンピューター）  → read as: コンピューター (computer)
インターネット（インターネット）  → read as: インターネット (internet)
```

## Reading Exercise: Decode the Hiragana

I'll show you sentences like this:

```
私（わたし）はプログラマーです。毎日（まいにち）コードを書（か）きます。
```

**How to read it:**

| Word | Kanji | Hiragana | Pronunciation | Meaning |
|------|-------|----------|---------------|---------|
| I | 私 | わたし | wa-ta-shi | I / me |
| (particle) | は | は | ha | topic marker |
| — | — | プログラマー | puro-gu-ra-mā | programmer |
| every day | 毎日 | まいにち | ma-i-ni-chi | daily |
| — | — | コードを | kō-do-o | code (object) |
| to write | 書 | か | ka | write |
| — | — | きます | ki-ma-su | (present tense) |

**Reading aloud:**
わたし は プログラマー です。まいにち コード を かき ます。

**English:** "I am a programmer. I write code every day."

## Pro Tips

1. **Don't memorize kanji first** — learn to read hiragana fluently first
2. **Sound out the hiragana slowly** — even if you don't know the kanji, you can pronounce it
3. **Particle hiragana** — tiny hiragana like は, を, に are *particles*, not content words
4. **Double consonants** — small つ (tsu) = pause/double the next sound
   - 学校（がっこう）= gak-kō (not gak-ko)
5. **Long vowels** — when hiragana repeats or uses う, it extends the sound
   - こうこう = kō-kō (high school) — the う extends the "o" sound
