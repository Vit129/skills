## Rule Structure

Each industry includes:

```yaml
- **Recommended Patterns**: 
  Primary layout strategies, component priorities

- **Style Priorities**: 
  Visual hierarchy (minimalism, bold, geometric, etc.)

- **Color Moods**:
  Psychological associations, industry norms, trust signals

- **Typography Personality**:
  Voice, readability, brand character

- **Key Effects**:
  Animations, transitions, micro-interactions

- **Anti-Patterns**:
  Common mistakes, what to avoid
```
