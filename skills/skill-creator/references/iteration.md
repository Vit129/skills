# Iteration Rules

When a skill isn't working:

| Problem | Fix |
|---------|-----|
| Agent doesn't follow the process | Edit SKILL.md — make the step clearer |
| Agent needs more info | Add a reference file |
| Agent does something unwanted | Add a rule ("NEVER...") |
| Agent can't use a tool | Teach manually once → create MCP reference doc |
| SKILL.md is cluttered | Move detail to references/ — keep only core routing |
