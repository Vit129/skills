# Data Gathering

## 2a) Qualitative (News, MD&A, Guidance)

Search queries:
```
[TICKER] latest earnings results [year]
[TICKER] management guidance outlook [year]
[TICKER] earnings call transcript [quarter year]
[TICKER] MD&A management discussion
[TICKER] recent news [month/year]
```

Extract: key themes from latest earnings call, forward guidance (quantitative targets if disclosed), risks/headwinds, strategic initiatives/M&A/product news, analyst sentiment.

## 2b) Revenue and Net Income (Last 5 Quarters)

```
[TICKER] quarterly revenue net income [year]
[TICKER] income statement quarterly
```

Collect per quarter: Revenue (absolute), Revenue YoY%, Revenue QoQ%, Net Income (absolute), Net Income YoY%, Net Income QoQ%. Label fiscal quarters clearly (e.g., Q1 FY2025).

## 2c) EPS (Last 5 Quarters)

```
[TICKER] EPS earnings per share quarterly
[TICKER] diluted EPS quarterly history [year]
```

Collect per quarter: Reported EPS, Consensus EPS (if available), Beat/miss label, YoY growth%, QoQ growth%.

Compute (analysis only): Momentum label (accelerating/decelerating/stable), Consistency label (improving/flat/deteriorating/volatile).

## 2d) Valuation

```
[TICKER] P/E ratio trailing forward
[sector] median P/E [year]
```

Collect: Trailing P/E (TTM), Forward P/E (NTM consensus, if available), Sector median (mark "approx." if uncertain), Premium/discount vs peers and sector.

## 2e) Dividend

```
[TICKER] dividend history yield payout ratio
[TICKER] dividend per share DPS [year]
```

Collect: DPS (most recent), Current yield, Payout ratio (if available), Consistency (cuts/increases). If none: note "No dividend".
